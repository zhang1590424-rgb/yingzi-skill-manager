# 21st Community Components

Source basis: official 21st community component pages, category pages, and the official sitemap.

Use this file for pragmatic UI primitives, community hero/layout ideas, shader experiments, and production-minded interface parts.

## High-level discovery pages

- [Community Components](https://21st.dev/community/components) - main discovery surface.
- [Popular Components](https://21st.dev/community/components/popular) - fast scan for stronger patterns.
- [Featured Components](https://21st.dev/community/components/featured) - curation surface.
- [Shader Components](https://21st.dev/community/components/s/shader) - shader-heavy hero, feature, and background patterns.
- [Shader Background For Hero](https://21st.dev/community/components/s/shader-bg-for-hero) - hero-specific shader discovery page.
- [Hover Effects](https://21st.dev/community/components/s/hover-effects) - hover interaction directory.
- [UI Layouts](https://21st.dev/community/ui-layouts) - popular scroll, gallery, carousel, and section ideas.
