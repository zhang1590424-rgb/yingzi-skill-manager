# Style Recipes

This file maps the shared components library to the specific taste skills. Use it to choose better-fitting component directions before inventing generic sections from scratch.

## Brutalism
- Start with: `gsap-explore.md` demos `directional-marquee`, `draw-a-path`, `card-stack`, `trigger-on-scroll`.
- Use from Aceternity: `background-boxes`, `layout-grid`, `focus-cards`, `container-scroll-animation` only after hardening them.
- Use from React Bits: `split-text`, `scrambled-text`, `count-up`, `card-nav`, `scroll-stack`.
- Use from 21st: pragmatic primitives only after the page skeleton is already strong.
- Push further by: larger image plates, chapter rails, manifesto slabs, hard CTA banners, and one strong reveal instead of many small gadgets.
- Avoid: glowing purple shader heroes, overloaded telemetry widgets, too many micro-hover tricks.

## Cinematic Product
- Start with: `gsap-explore.md` demos `image-sequence`, `flip-gallery-modal`, `horizontal-scrolling-gallery`, `responsive-line-splits-on-scroll`, `image-mask-on-scroll`.
- Use from Aceternity: `hero-parallax`, `apple-cards-carousel`, `container-cover`, `3d-pin`, `compare`.
- Use from React Bits: `split-text`, `text-type`, `glare-hover`, `orbit-images`, `fade-content`.
- Use from 21st: hero gradient references, card carousel, image gallery, shader hero patterns.
- Push further by: hero stills, product renders, chapter-based reveals, premium stat blocks, and stronger top-rail navigation.
- Avoid: generic dark startup gradients, tiny product screenshots, and bento grids as the main hero.

## Dark Luxe
- Start with: `gsap-explore.md` demos `text-masking`, `smooth-scrolling`, `image-mask-on-scroll`, `velocity-skew`.
- Use from Aceternity: `hero-highlight`, `lamp-effect`, `card-spotlight`, `glare-card`, `compare`.
- Use from React Bits: `shiny-text`, `metallic-paint`, `glare-hover`, `shape-blur`, `fade-content`.
- Use from 21st: selective gradient/shader references only if they are desaturated and luxurious.
- Push further by: large campaign photography, light-sculpted product crops, subtle reveal masks, and refined nav treatment.
- Avoid: glittery purple-blue effects, over-ornamented cards, and too many tiny luxury badges.

## Dashboards
- Start with: `gsap-explore.md` demos `animate-auto-height`, `animate-between-layout-positions`, `animate-css-grid-positions`, `trigger-on-scroll`, `animate-scroll-position`.
- Use from Aceternity: `expandable-card`, `compare`, `layout-grid`, `file-upload`, `animated-tooltip`.

- Use from 21st: progress, breadcrumb, select, dropdown, textarea, card, carousel, hover-card.
- Push further by: diagrams, charts, maps, process visuals, row highlights, count-up reveals, and cleaner toolbar systems.

## Gallery Minimal

## Minimalism
- Start with: `gsap-explore.md` demos `animate-text`, `animate-auto-height`, `trigger-on-scroll`, `enter-and-exit`.
- Use from Aceternity: `hero-highlight`, `background-lines`, `compare`, `hover-border-gradient` very lightly.
- Use from React Bits: `split-text`, `blur-text`, `fade-content`, `count-up`.
- Use from 21st: interface primitives for forms and buttons when the page needs quiet utility.

- Avoid: empty text stacks, excessive ornament, and loud motion that breaks restraint.
