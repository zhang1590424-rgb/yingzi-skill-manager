# 影子产品介绍视频设计

## 背景

素材为 `录屏2026-05-25 下午9.41.59.mov`，时长 47 秒，内容覆盖「影子」从初始化配置到 Skill 应用状态展示的完整操作流。

本次目标不是教程，而是剪成一条 **20 秒快节奏产品介绍视频**，让不了解项目的人快速理解：

- 影子是本地 Skill 管理器。
- 它把 Skill 主库分发到不同 Agent 和项目。
- 用户能看见未应用、已应用和目标状态。

## 剪辑方向

采用「冷静产品感」：

- 保留真实应用界面作为主体，避免重绘假界面。
- 使用局部放大、轻微推拉、快速转场提升节奏。
- 字幕解释价值点，不把画面变成说明书。
- 视觉沿用项目 `DESIGN.md`：浅灰工作台、近黑主文字、低饱和状态色、小圆角、细边框，不使用紫色渐变、发光边框、玻璃拟态或广告式大字轰炸。

对用户的影响：成片看起来像一个真实可用的本地工具，而不是概念宣传页；首次观看者能在短时间内建立“它解决 Skill 分发和状态确认”的认知。

## 成片结构

| 时间 | 画面 | 字幕 | 目的 |
| --- | --- | --- | --- |
| 0.0s-2.0s | 应用设置 / Agent 列表画面快速入场 | 影子 | 建立产品名和桌面工作台气质 |
| 2.0s-5.0s | 添加项目、选择本地目录 | 接入 Agent 和项目 | 说明管理范围来自本地路径 |
| 5.0s-8.5s | 预置 Skill 安装列表 | 先把 Skill 放进主库 | 建立“主库是事实源” |
| 8.5s-12.5s | 分发抽屉 / 多 Agent 目标列表 | 一次分发到多个目标 | 解释核心动作是软链接分发 |
| 12.5s-17.0s | Agent / 项目应用双栏状态 | 未应用、已应用，一眼确认 | 强调状态可见、可验证 |
| 17.0s-20.0s | 双栏结果画面 + 收束标题 | 本地 Skill 管理，收放有据 | 总结价值点 |

## 画面处理

- 画幅：16:9，1920x1080。
- 原录屏比例约 1.49:1，放入 16:9 画布时使用浅灰背景和窗口投影承托，不强行裁掉关键 UI。
- 每段画面做 1.03-1.10 倍轻推拉；关键状态区域使用短时局部放大。
- 转场使用快速滑移或遮罩扫过，保持工作台克制感。
- 字幕控制在每屏 8-12 个中文字符，避免遮挡界面核心区域。
- 不保留原声；输出为无旁白版本，可后续叠加音乐或配音。
